# AirCanvas – Hand Gesture Drawing System

✋ AirCanvas is a virtual drawing application that allows users to draw on a digital canvas using hand gestures. It leverages computer vision to track hand movements in real time, enabling a touch-free drawing experience.

---

## ✨ Features
- 🖐️ Real-time hand gesture tracking  
- 🎨 Draw on screen without physical contact  
- 🖍️ Multiple drawing tools (pen, shapes, eraser)  
- 🎙️ Voice feedback integration  
- ⚡ Smooth and responsive drawing experience  

---

## 🛠️ Tech Stack
- Python  
- OpenCV  
- MediaPipe
